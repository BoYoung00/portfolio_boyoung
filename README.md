### 📌 주요 기능

#### 1. 움직이는 펭귄
   - MouseEvent를 통해 펭귄 아이콘이 클릭한 위치로 이동


#### 2. 데이터 입력 및 변환
   - IntersectionObserver을 통해 스크롤 애니메이션 


#### 3. 스킬 레벨바 애니메이션
  - ref로 하위 컴포넌트의 애니메이션을 제어
